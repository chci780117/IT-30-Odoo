 |  屬性 | 內容 | 
 | -------- | -------- | 
 | id | 定義了一個 XML ID，可讓其他記錄來引用它。| 
 | name  | 僅供參考；它不必是唯一的，這個名字可以完全省略。忽略的話 Odoo 會從 模型名稱 和 視圖類型 自動產生 name。 |  
 | model | 指名關聯哪個模型 |  
 | arch  | 視圖定義 |
